## Desc
This project is called open-e-learn, which aims to create an e-learning system for students with the following business flows:
### Students

- Students can register for courses by filling out the KRS (Course Plan) and no longer need to register manually.
- Students automatically drop out of courses when the semester ends.
- Students can view course materials (in .pdf/ppt/docs/etc. files) and a list of course assignments. Students can only view course materials if they are enrolled in the course.
- Students can remind their lecturers to upload course materials that have been explained in class.
- Students can consult with lecturers online.
- Students can view the class schedule, not just in PDF format, but also with a user interface (UI/UX) that makes it easy for them to view the class schedule. Students can only view the class schedule for the courses they are taking.

### Lecturers

- Lecturers can teach specific courses selected before the semester begins. This means that each semester, lecturers can teach different courses. This can be done by sending a PDF/Excel file, which will then be extracted to determine which lecturer is teaching a particular course.
- Lecturers can upload course materials and assignments, but only if the lecturer is the course instructor. Students will receive an email notification after uploading.

### Admin
- Admins can create class schedules by uploading PDF files from the campus.
